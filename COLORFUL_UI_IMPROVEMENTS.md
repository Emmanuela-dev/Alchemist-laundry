## 🌈 Gradient Usage

### Why Gradients?
1. **Modern**: Current design trend
2. **Depth**: Creates visual interest
3. **Premium**: Elevates perceived value
4. **Distinctive**: Memorable brand identity

### Gradient Rules
- Always 2 colors max per gradient
- Lighter color on bottom-right
- 45° angle (topLeft → bottomRight)
- Smooth transitions
